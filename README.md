# T20 World Cup Data Analysis — Python, Matplotlb & Seaborn

This project explores match-level insights from the T20 World Cup using Python, Pandas, and Seaborn. It analyzes toss decisions, venue popularity, match outcomes, and player performance to uncover patterns in tournament dynamics.

---

## Project Highlights

- Cleaned and explored match-level data from the T20 World Cup
- Visualized toss decisions and their impact on match outcomes
- Analyzed venue frequency and match distribution
- Identified top performers based on "Man of the Match" awards
- Categorized win types (runs vs wickets) and visualized their distribution
